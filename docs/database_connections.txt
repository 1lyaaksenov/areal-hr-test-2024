Organizations Departments:

связь: Один ко многим.
описание: Каждая организация (organizations) может иметь несколько отделов (departments), но каждый отдел принадлежит только одной организации.

Departments Employees:

связь: Один ко многим.
описание: Каждый отдел может иметь несколько сотрудников, но каждый сотрудник относится только к одному отделу.

Status Employees:

связь: Один ко многим.
описание: Каждый статус может быть назначен нескольким сотрудникам, но у сотрудника может быть только один статус(уволен или нет).

Employees Files:

связь: Многие ко многим.
описание: Один сотрудник может иметь несколько прикрепленных файлов, но каждый файл относится только к одному сотруднику.

Employees hr_operations:

связь: Один ко многим.
описание: Один сотрудник может иметь множество HR-операций, но каждая HR-операция привязана к конкретному сотруднику.

Positions Employees:

связь: Один ко многим.
описание: На каждой должности может быть несколько сотрудников, но у сотрудника только одна должность.
